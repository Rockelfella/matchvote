2. Erhebung und Verarbeitung personenbezogener Daten

2. Data Collection and Processing

Bei der Nutzung der MatchVote-App werden nur solche Daten erhoben, die für den Betrieb der Plattform technisch erforderlich sind. Dazu gehören insbesondere:

When using the MatchVote app, only data that is technically necessary for operating the platform is collected. This may include:

  • technisch notwendige Zugriffsdaten
  • anonymisierte Nutzungs- und Bewertungsdaten
  • bei Registrierung: E-Mail-Adresse (pseudonymisiert gespeichert)
  • technically required access data
  • anonymized usage and rating data
  • during registration: email address (stored in a pseudonymized form)

Eine Zuordnung von Bewertungen zu realen Personen findet nicht öffentlich statt.

Ratings are not publicly linked to identifiable individuals.

3. Zweck der Datenverarbeitung

3. Purpose of Data Processing

Die Verarbeitung der Daten erfolgt ausschließlich zu folgenden Zwecken:

Data is processed solely for the following purposes:

  • Bereitstellung und Betrieb der App
  • Speicherung und Aggregation von Nutzerbewertungen
  • Sicherstellung der technischen Stabilität und Sicherheit
  • providing and operating the app
  • storing and aggregating user ratings
  • ensuring technical stability and security

4. Weitergabe von Daten

4. Data Sharing

Es erfolgt keine Weitergabe personenbezogener Daten an Dritte zu Werbe- oder Marketingzwecken.

Personal data is not shared with third parties for advertising or marketing purposes.

5. Speicherung und Löschung

5. Data Retention

Personenbezogene Daten werden nur so lange gespeichert, wie dies für den jeweiligen Zweck erforderlich ist oder gesetzliche Aufbewahrungspflichten bestehen.

Personal data is stored only for as long as necessary to fulfill the respective purpose or as required by law.

Users have the right to request access to their personal data, as well as rectification, deletion, restriction of processing, and data portability.

Anfragen können jederzeit per E-Mail an info@matchvote.de gerichtet werden.

Requests can be sent at any time to info@matchvote.de.
